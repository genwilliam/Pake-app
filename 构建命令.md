# Mac

以下为 macOS 下的构建命令，均为可直接执行的 `pake` 命令。

如果你没有pake命令,请你查看 [Pake 官方文档](https://github.com/tw93/Pake/blob/main/docs/cli-usage.md)

## Bilibili

```bash
pake https://www.bilibili.com \
	--name "Bilibili" \
	--width 1400 \
	--height 900 \
	--hide-title-bar \
	--force-internal-navigation \
	--new-window \
	--proxy-url socks5://127.0.0.1:6153
```
